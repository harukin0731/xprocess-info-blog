# このサイトについて

## Xprocess とは

XProcess は、harukin の作成物を公開する際の名義です。アプリケーションやソフトウェアの公開、管理、サポートなどを行うための名義となります。
このサイトは、Xprocess のアプリケーションやソフトウェアの更新情報を公開するためのものです。
コンテンツ本体は、[harukin の web サイト](https://haruk.in)にて公開しております。
